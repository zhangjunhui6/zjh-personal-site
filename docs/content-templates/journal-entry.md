# 生活记录模板

适用于 `src/content/journal`。

```md
---
title: 记录标题
description: 用一句话描述这次记录留下了什么感觉或观察。
date: YYYY-MM-DD
tags: [生活, 观察]
lang: zh
draft: true
mood:
location:
images: []
media: []
---

用一段话进入现场：时间、地点、状态或一个细节。

## 发生了什么

写事情本身，不需要太完整，但要保留能让以后重新进入记忆的细节。

## 当时注意到的东西

- 观察一。
- 观察二。
- 观察三。

## 后来想到的

写这件事带来的想法。可以松一点，不必像技术文章一样完整论证。

## 留下来的一句话

用一句话收束这次记录。
```
